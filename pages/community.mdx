# Community

### On Discord

Join our [Discord community](https://discord.orbit.love/) of Orbit and Orbit Model users.
Visit the _#orbit-model_ channel for discussions about the model itself.

### On GitHub

Visit the Orbit Model [GitHub repository](https://github.com/orbit-love/orbit-model)
to learn how to contribute.

### Via email

Have a question, an idea, or a topic you'd like to discuss? Please
email [model@orbit.love](mail:model@orbit.love).

## Contribute

Here are some of the types of contributions we're looking for.

| Contribution Type                       |
| --------------------------------------- |
| Bringing the model to new domains       |
| Advocating via talks and blog posts     |
| Suggesting new ideas and new techniques |
| Being interviewed for case studies      |
| Improving documentation                 |
| Translating documentation               |

If you're interested in doing any of these, please reach out via email or
Discord. Thanks!

## Areas of interest

If you're doing academic research in any of the following areas and
feel like there is a tie-in between your research and the Orbit Model,
we'd particularly love to hear from you.

- Social network analysis
- Sociology
- Artificial Intelligence
- Anthropology

## Code of Conduct

Please review the Orbit Model community [Code of
Conduct](https://github.com/orbit-love/orbit-model/blob/main/CODE_OF_CONDUCT.md).
